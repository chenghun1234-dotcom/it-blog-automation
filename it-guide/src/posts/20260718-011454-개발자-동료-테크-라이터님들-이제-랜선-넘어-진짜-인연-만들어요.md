# 개발자 동료, 테크 라이터님들! 이제 랜선 넘어 '진짜' 인연 만들어요!

안녕하세요, DEV 커뮤니티 여러분! 저는 토보레입니다. 그동안 DEV 커뮤니티에서 글도 쓰고 다른 분들 포스팅도 조용히 지켜봐 왔지만, 문득 실제로는 많은 분과 깊이 연결되지 못하고 있다는 생각이 들었어요. 그래서 이 기회를 빌려 이 간극을 메워보려 합니다.

저는 나이지리아 포트하코트에서 활동하고 있습니다. 주로 웹사이트 개발과 함께 테크니컬 문서 작성, 콘텐츠 라이팅 업무를 맡고 있죠. 제가 쓰는 글 대부분은 단순히 어딘가에서 읽은 지식을 나열하는 것이 아니라, 실제로 제가 직접 만들고 배포했던 프로젝트 경험에서 나옵니다. 현재 DEV 커뮤니티의 React Top Author 배지를 보유하고 있으며, 최근에는 Google I/O 2026 Technical Writers Writing Challenge에서 우승하는 영광을 안기도 했습니다.

현재는 제 커리어를 탄탄하게 쌓아가는 데 집중하며, 프리랜서 개발 및 라이팅 프로젝트를 진행하고 있어요. 틈틈이 AWS SAA-C03 자격증 취득을 위해 공부도 병행하고 있고요. 또한, 'BuildWithTee'라는 프로젝트를 운영 중인데, 이는 코드를 짜고, 글을 쓰고, 연결되는 과정을 공개적으로 보여주려는 제 시도입니다.

제가 실무에서 여러 프로젝트를 진행하고 AWS 같은 클라우드 환경을 다루면서 느낀 점은, 단순히 기술을 잘 아는 것을 넘어 '어떻게 공유하고 연결하느냐'가 정말 중요하다는 겁니다. 특히 'Build in Public' 같은 시도는 단순히 개인 브랜딩을 넘어 학습 과정을 가속하고 예상치 못한 협업 기회를 가져다주더군요.
