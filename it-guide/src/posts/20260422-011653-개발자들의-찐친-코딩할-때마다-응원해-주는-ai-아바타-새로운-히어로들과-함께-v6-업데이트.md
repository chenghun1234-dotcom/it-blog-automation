# 개발자들의 찐친! 코딩할 때마다 응원해 주는 AI 아바타, 새로운 히어로들과 함께 v6 업데이트!

그동안 인공지능과 대화하거나 엔터를 누를 때마다 반응하는 VRM 3D 캐릭터, 즉 저만의 AI 아바타를 만들고 꾸준히 업데이트해왔습니다. 처음부터 기본 아바타는 '닌자 걸' 한 명이었죠. 물론 직접 만든 아바타로 변경할 수도 있었지만, 여러 기본 아바타가 각자의 애니메이션과 스토리를 가지고 있다면 훨씬 더 재미있을 거라고 생각했어요. 저도 실무에서 한 가지 아바타만 쓰다 보니 가끔은 새로운 분위기로 기분 전환이 필요하다는 생각을 했거든요. 그래서 이번엔 동료 'Maximum Blue' 님이 아바타 제작을 흔쾌히 도와주셨습니다. 정말 감사드립니다!

## 버전 6 업데이트: Joyful Colors, 당신의 활력소가 되다!

이번 **버전 6 업데이트**의 핵심은 바로 '개발자와 모든 이들을 행복하게 응원한다'는 미션으로 똘똘 뭉친 그룹, **Joyful Colors**입니다. 이제 이 친구들이 VS Code 확장 프로그램과 크롬(엣지) 확장 프로그램을 통해 하루 24시간 내내 여러분 곁에서 힘을 불어넣어 줄 거예요. 그럼 우리의 새로운 히어로 캐릭터들을 한번 만나볼까요?

## 버전 5 업데이트: 더 부드럽고 편리하게!

물론, 버전 6 업데이트 이전에 몇 가지 중요한 개선 사항, 버그 수정 및 조정을 진행했습니다.

*   **손가락에 뼈대(Bone)를 추가했어요.** 이전 버전에서는 아바타 손이 항상 펼쳐져 있었죠. 이제는 브이(V) 자를 그리거나 주먹을 쥐는 등 훨씬 더 자연스러운 손동작을 표현할 수 있게 되어 아바타가 훨씬 생동감 있게 느껴집니다.
{% youtube prPDPsb2ZKM %}

*   **애니메이션 값을 라디안(예: 1.57)에서 각도(예: 90)로 변경하여 직관적인 이해를 돕도록 했습니다.** 제가 직접 애니메이션 값을 조정할 때 라디안 단위는 직관성이 떨어져서 불편했거든요. 기존 파일과의 하위 호환성도 물론 유지됩니다.

*   **배경색을 회색으로 추가했습니다.** 아바타가 검은색 장갑을 끼고 있을 때 검은색 배경에 묻혀서 잘 보이지 않는 문제가 있었죠. 이제 사용자는 검은색 또는 회색 배경 중 원하는 색상을 선택할 수 있습니다.

*   VS Code 터미널에서 **Enter 키를 눌렀을 때도 애니메이션이 트리거되도록 수정했습니다.** 덕분에 훨씬 더 다양한 상황에서 아바타의 재미있는 반응을 즐길 수 있게 되었죠.
