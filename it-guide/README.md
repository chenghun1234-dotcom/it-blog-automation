# IT 가이드 포털

지능형 IT 가이드 포털 - Neo-Tech Minimal 디자인을 적용한 현대적인 웹사이트.

## 디자인 콘셉트

- **UI 디자인**: "Neo-Tech Minimal"
- **핵심 철학**: 정보는 명확하게, 배경은 몽환적으로, 상호작용은 매끄럽게.
- **시각적 특징**:
  - Glassmorphism: 반투명 배경과 블러 효과
  - Bento Grid Layout: 크기가 다른 사각형 격자
  - Micro-interactions: 호버 효과와 애니메이션
  - Color Palette: 다크 그레이/블랙 배경, 네온 블루/퍼플/그린 강조색

## 기술 스택

- **프레임워크**: Next.js 16 (App Router)
- **스타일링**: Tailwind CSS
- **애니메이션**: Framer Motion
- **언어**: TypeScript

## 시작하기

개발 서버 실행:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

## 빌드

프로덕션 빌드:

```bash
npm run build
```
